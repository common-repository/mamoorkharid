mamoorkharid search engine
موتور جستجوی سامانه‌ی مأمور خرید
== Description ==
این افزونه تمامی کارهای روزمره خرید و فروش شما را به نحواحسنت انجام میدهد
از انتخاب بهترین کالا یا خدمات از هزاران عرضه کننده مرکزبازار تا رسیدن به بهترین قیمت توسط مامورخرید و عرضه مستقیم کالا یا خدمات
کار این افزونه جستجو در مشاغل و خدمات سایت مامورخرید به آدرس https://mamourkharid.com/ میباشد

this plugin search in jobs and services in site https://mamourkharid.com/ and other site related.
the related site added step by step

== Installation ==
1. افزونه را یا مسقیما آپلود کنید و یا از طریق مخزن وردپرس آنرا نصب نمایید
2. با استفاده از شورتکدهای ارائه شده در بخش راهنمای افزونه میتوانید آنرا در هر قسمت از سایت قرار دهید
